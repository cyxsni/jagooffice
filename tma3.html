Belajar Microsoft Access: Membuat Tabel



Tabel di Microsoft Access adalah struktur yang menyimpan data dalam bentuk baris dan kolom, di mana kolom mewakili atribut dan baris mewakili record. Tabel digunakan untuk mengorganisir dan mengelola informasi dalam basis data.

Sebelumnya, kita buat dulu databasenya. di Microsoft Accesse

Langkah 1

  • Buka Aplikasi Microsoft Access
  • Klik File New
  • Pilih Blank Database

Langkah 2

Create nama file database

Untuk membuat tabel, klik tab create pilih table

  • Table adalah dasar dari database Microsoft Access. Di sini kamu menyimpan data dalam bentuk kolom (field) dan baris (record).
  • Table Design adalah tempat kamu mendefinisikan struktur tabel, termasuk jenis data yang disimpan, validasi data, dan hubungan dengan tabel lain.
  • SharePoint Lists adalah fitur di platform SharePoint yang memungkinkan kamu membuat daftar dan menyimpan data dengan cara yang sederhana.
