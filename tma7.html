Belajar Microsoft Access: Cara Membuat Report



Report merupakan fasilitas dalam Microsoft Access yang berfungsi untuk mencetak data dalam bentuk laporan. Dengan report, kita menjadi lebih leluasa dalam membuat laporan. Data yang digunakan untuk membuat laporan, dapat berasal dari tabel maupun query.

Untuk membuat report, klik tab create pilih report/report design/blank report

  • Report adalah hasil akhir dari proses pembuatan laporan di Microsoft Access. Report menampilkan data dalam format yang lebih terstruktur dan informatif dibandingkan dengan tabel.
  • Report Design adalah mode desain untuk membuat laporan baru. Di sini kamu dapat mengatur layout, menambahkan elemen seperti teks, gambar, dan kontrol lainnya.
  • Blank Report adalah template kosong untuk membuat laporan baru dari awal.
  • Report Wizard membantu kamu membuat laporan dengan cepat dengan panduan langkah demi langkah. Wizard akan menanyakan jenis data yang ingin kamu tampilkan dan format laporan yang diinginkan.
  • Labels adalah form khusus yang digunakan untuk mencetak label, seperti label alamat atau label produk.
